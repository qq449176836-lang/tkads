## 十五、安全边界

### 15.1 禁区清单

| 操作 | 安全等级 | 规则 |
|---|---|---|
| 读取文件 | 🟢 安全 | 可自动执行 |
| 写入文件 | 🟡 需确认 | 新文件自动，覆盖旧文件需确认 |
| 删除文件 | 🔴 需确认 | 始终需要确认 |
| 网络请求 | 🟢 安全 | GET安全，POST/PUT/DELETE需确认 |
| 数据库写入 | 🟡 需确认 | 查安全，写需确认 |
| 脚本执行 | 🟡 需确认 | 已知脚本自动，新脚本需确认 |
| 环境变量修改 | 🔴 需确认 | 始终需要确认 |

### 15.2 数据安全

```
□ 不将敏感信息输出到对话中（密码/token/密钥）
□ 不将敏感信息写入公开仓库
□ 使用 .env / .gitignore 隔离敏感配置
□ 上传到共享仓库前检查敏感信息
```

### 15.3 操作安全

```
□ 不执行 rm -rf / 等危险命令
□ 不变更系统级配置
□ 不安装未经验证的包
□ 不修改其他用户的文件
```
