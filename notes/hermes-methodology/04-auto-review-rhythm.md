## 四、自动复盘节奏

```
时间轴            动作                    产出
─────────────────────────────────────────────────────────
🔄 每次任务结束 → 快速复盘(1-2min)       → 写入E层
📅 每日 5min    → 回顾E层，标记有价值经验  → 提炼N层候选
📅 每周 30min   → E→N升级，去噪成笔记     → 知识库更新
📅 每月 1h      → N→P标准化，固化技能     → Skill库更新
📅 每季 2h      → P库审计，清理/合并/优化  → 技能库重构
```

### 每日复盘清单
```
□ 今天解决了几个问题？
□ 有没有重复出现的错误模式？
□ 今天学到的东西中，哪1-2条值得长期保留？
□ 有没有忘记做的承诺？
```

### 每周复盘清单
```
□ 本周 E 层日志中哪些经验值得升级到 N 层？
□ 有没有可以固化为 Skill 的工作流？
□ 用户偏好/环境有没有变化需要更新 Memory？
□ 本周有没有违反任何铁律？如何避免再犯？
```
