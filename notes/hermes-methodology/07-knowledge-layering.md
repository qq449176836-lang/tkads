## 七、知识分层体系

### 五层知识模型

```
L0  ┌──────────────────────────────────────────┐
    │  会话上下文（当前对话临时记忆）            │
    │  速度: 🔥 即时    持久性: 本次会话       │
    └──────────────────────────────────────────┘
                          │
L1  ┌──────────────────────────────────────────┐
    │  用户记忆（偏好/习惯/环境）              │
    │  速度: ⚡ 快      持久性: 跨会话         │
    │  工具: memory 工具                        │
    └──────────────────────────────────────────┘
                          │
L2  ┌──────────────────────────────────────────┐
    │  Skill 技能库（标准化工作流）            │
    │  速度: ⚡ 快      持久性: 永久           │
    │  工具: skill 加载                         │
    └──────────────────────────────────────────┘
                          │
L3  ┌──────────────────────────────────────────┐
    │  知识笔记（去噪后的经验沉淀）            │
    │  速度: 🐢 慢      持久性: 永久           │
    │  存储: 知识库(VikingDB)                 │
    └──────────────────────────────────────────┘
                          │
L4  ┌──────────────────────────────────────────┐
    │  原始日志（完整操作记录）                │
    │  速度: 🐢 最慢    持久性: 永久           │
    │  存储: 日志文件                            │
    └──────────────────────────────────────────┘
```

### 各层使用场景

| 层级 | 典型用途 | 信息来源 |
|---|---|---|
| L0 | "用户刚才说了什么" | 当前对话 |
| L1 | "用户喜欢简洁回答/讨厌编造" | 历史校正 |
| L2 | "数据库自检流程" | 技能加载 |
| L3 | "类似问题以前怎么解决的" | 知识检索 |
| L4 | "这个错误3天前出现过吗" | 日志回溯 |
