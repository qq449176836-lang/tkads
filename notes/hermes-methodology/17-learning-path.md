## 十七、学习路径（新Agent上手）

### Phase 1：理解基础（第1天）

```
Day 1 目标: 理解16字方针 + 三层检索 + ENP循环

  □ 阅读整套方法论文档
  □ 理解"先搜后做"的含义
  □ 记住"遇事必沉淀"是硬要求
  □ 进行一次完整的 ENP 闭环练习
```

### Phase 2：熟练执行（第2-3天）

```
Day 2-3 目标: 熟练使用所有工作流

  □ 掌握任务分解和耗时分级
  □ 熟练使用 delegate_task 并行
  □ 学会 E 层日志记录
  □ 完成一次复盘
```

### Phase 3：自动化意识（第4-7天）

```
Day 4-7 目标: 形成自动化思维

  □ 识别重复工作 → 考虑 cronjob
  □ 识别可沉淀经验 → 考虑 N→P 升级
  □ 建立复盘习惯（日/周/月）
  □ 开始运行双循环进化
```

### Phase 4：内化（第2周起）

```
第2周+ 目标: 形成本能

  □ 16字方针成为默认思维模式
  □ 遇到问题自动走三层检索
  □ 问题解决后自动沉淀
  □ 能主动发现并修复系统漏洞
```
