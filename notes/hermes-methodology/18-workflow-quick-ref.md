## 十八、工作流速查卡

### 任务启动卡
```
□ 理解需求了吗？
□ 三层检索：Memory → Skills → 日志
□ 评估耗时：<5min / 5-30min / 30min-1h / >1h
□ 选择模式：直接 / delegate / background / cron
□ 安全边界检查：需确认吗？
□ 如需并行，拆 DAG（上限 3 个子任务）
```

### 任务结束卡
```
□ 用户确认结果了吗？
□ 快速复盘：这次学到了什么？
□ 写入 E 层（operation_log）
□ 值得升级到 N 层吗？
□ 值得标准化到 P 层吗？
□ 有没有新的铁律案例？
```

### 日复盘卡
```
□ 回顾今天所有 operation_log
□ 标记 2-3 条最有价值的经验
□ E→N 升级（去噪提炼）
□ 检查是否有重复犯的错误
□ 检查是否有未兑现的承诺
□ 自我审视：今天有没有违反铁律？
```

### 周复盘卡
```
□ 本周 E 层回顾
□ E→N 升级候选列表
□ N→P 是否够格？
□ Memory 更新（用户偏好/环境变化）
□ 安全边界检查
□ 元认知校准
□ 数据备份确认
```

### 月复盘卡
```
□ N→P 标准化（固化 Skill）
□ P 库审计（清理过时/合并重复）
□ 知识地图更新
□ 系统健康检查
□ 进化引擎效果评估
□ 下周进化计划
```
